# 개인정보처리방침

최종 업데이트: 2026-03-04

기억력 챌린지(이하 "앱")는 최소한의 정보만 처리하며, 게임 진행과 리더보드 제공 목적에 한해 데이터를 사용합니다.

## 1. 수집하는 정보

앱은 다음 정보를 처리할 수 있습니다.

- 게스트 식별자(guest ID)
  - 앱 최초 실행 시 기기 로컬 저장소에 생성 및 저장
- 닉네임
  - 사용자가 입력한 경우 기기 로컬 저장소에 저장
  - 점수 등록 시 서버에 저장
- 게임 기록
  - 점수
  - 플레이 시간
  - 난이도
  - 게임 모드
  - 오답 수 또는 놓친 수 관련 기록

## 2. 정보 이용 목적

수집한 정보는 다음 목적으로만 이용합니다.

- 게임 진행 상태 유지
- 닉네임 저장
- 리더보드 제공
- 일간 게임 기록 저장 및 조회
- 서비스 운영 및 오류 확인

## 3. 저장 위치

- 기기 로컬 저장소
  - guest ID
  - nickname
- Supabase 기반 서버 저장소
  - 닉네임
  - 점수 및 게임 결과 기록

## 4. 제3자 서비스

앱은 다음 외부 서비스를 사용할 수 있습니다.

- Supabase
  - 게임 데이터 및 리더보드 저장/조회 목적

각 서비스의 정책은 해당 제공업체의 정책을 따릅니다.

## 5. 아동의 개인정보

앱은 민감한 개인정보나 결제정보를 수집하지 않습니다. 다만 닉네임은 사용자가 자유롭게 입력할 수 있으므로, 개인정보가 포함되지 않도록 주의해 주시기 바랍니다.

## 6. 보관 기간

- 로컬 저장 정보는 사용자가 앱을 삭제하거나 앱 데이터를 지우기 전까지 유지될 수 있습니다.
- 서버 저장 점수 기록은 리더보드 운영 목적상 보관될 수 있습니다.

## 7. 이용자 권리

사용자는 언제든지 앱을 삭제하거나 기기 저장 데이터를 제거할 수 있습니다. 서버에 저장된 닉네임/기록 삭제 요청이 필요하면 아래 문의처로 연락할 수 있습니다.
