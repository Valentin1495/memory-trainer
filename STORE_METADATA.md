2회 연속 정확도가 낮으면 난이도를 낮춰주고,

매일 훈련을 완료하면 연속 일수가 쌓입니다.

## Resolution Center 답변 초안 (심사 탈락 시)

This app is a personalized memory training platform, not a generic memory game.
Key differentiating features:

1. Initial diagnostic test (3 levels: Easy/Medium/Hard) to measure baseline memory performance
2. Rule-based personalized training recommendation based on session history
3. Missed word tracking and weak-point analysis after every session
4. Weekly training report with bar chart (frequency), accuracy trend, and Top 5 weak words
5. Streak counter and daily training goal to support habit formation
6. Offline-first architecture with local category fallback

The app's home screen is a training dashboard (not a game menu), and the primary user flow is:
Onboarding → Diagnosis → Dashboard → Training → Session Analysis → Weekly Report

These features collectively provide a structured, personalized brain training experience
that is fundamentally different from single-session memory games on the App Store.
